Usage and Syntax

  1. Create Table 
    2. create table <table_name> (<column_name> <datatype> ,<column_name> <datatype> ... )
  2. Insert Record
    3. insert into <table_name> (<column_name>,<column_name> ...) values (<value>, <value> ...)

    To add entries in columns by default: 

    insert into <table_name> values (<value>, <<>value> ...)

  3. Delete records
    4. insert into <table_name> (<column_name>,<column_name> ...) values (<value>, <value> ...)
  4. Select records
    5. select <column_name> as <alias>, <column_name> as <alias> ... from <table_name> where <value_expession> order by <value_expession>, <value_expession> ...
    select <column_name> <alias>, <column_name> <alias> ... from <table_name> where <value_expession> order by <value_expession>, <value_expession> ...

    To select all columns: 

    select * from <table_name> where <value_expession> order by <value_expession>, <value_expession> ...<

  5. Displaying the whole database instance
    6. output
  6. Exiting:
    7. exit
